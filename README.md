# CSCI 1300 Final Project

## Theme

Fishing. The best part of the game.

## Goal

To fish and catch all legendary fish

## How to compile and run

Using a cpp file

## How to play

Text based in a day system. You fish for both money and for food to manage your hunger, you use moeny to upgrade oyur fishign rod to get better fish and to catch the things needed for the community center.

## Classes

Item: Holds the item name, if it can be sold, how much is it, and is it legendary
Bundle: Keeps track of the community center bundle and its progress
Player: Keeps track of the systems the player has to manage and features in the menu
Game: The main part of the code with all of the gameplay inside
